//Вступление

Приложение используется для поиска и просмотре информации о фильмах.
Приложение построенно на трехуровневой архетектуре MVC(Model-View-Controller).

//Развертывание

Для корректного запуска приложения понадобиться Microsoft SQL Server 2017(Если версия отличается, меняйте строку подключения),
Microsoft Visual Studio 2017 с установкой дополнительных пакетов GuNet:
-	Entity framework.
-	Bootstrap 4.
-	jQeury.
-	Popper.js.

//Описание

Существует 4 модели:
-	Актеры.
-	Фильмы.
-	Модель добавления данных в базу данных.
-	Модель создания констекста базы данных.

Один конроллер, отвечающий за вывод и поиск фильмов. Поиск осуществляется по названию фильма, жанру и имени актера. 

Одно представление, в котором отображается основной контент приложения, а так же частичное представление для реализации поиска.

Файл базы данных сохраняется после остановки сервера и перезаписывается каждый при добавление, изменение и удаление информации.
Таблица Films и таблица Actors связынны многие ко многим.
